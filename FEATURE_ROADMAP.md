# Recipe App - Feature Roadmap

## High Priority (Core Functionality)

- [ ] **Favorites/Rating System** - Star recipes to quickly find favorites; rate recipes (1-5 stars) to remember which ones you enjoyed most.

- [x] **Shopping List Generator** - Select recipes and auto-generate an aggregated shopping list with consolidated ingredients (e.g., if 2 recipes need butter, combine quantities). Export as text/PDF.

- [x] **Recipe Editing & Deletion** - Ability to modify existing recipes, fix typos, or remove recipes entirely. Currently you can only add them.

- [ ] **Dietary Filters** - Add flags for vegetarian, vegan, gluten-free, dairy-free, etc. Filter by these attributes.

- [ ] **Serving Size Scaler** - Automatically adjust ingredient quantities when changing servings (e.g., serve 4 instead of 2).

## Medium Priority (Quality of Life)

- [ ] **Nutrition Information** - Track calories, protein, carbs, fats per serving (could partially auto-populate from URL import).

- [ ] **Difficulty Level** - Mark recipes as easy/medium/hard for quick meal planning.

- [ ] **Recipe Notes/Reviews** - Add personal notes about what you changed, whether you'd make it again, when you last made it.

- [ ] **Seasonal/Occasion Tags** - Mark recipes as summer, holiday, weeknight, guest-friendly for event planning.

- [ ] **Cook Time Filters** - Filter by prep + cook time to find quick meals when in a hurry.

## Enhancement Ideas

- [ ] **Meal Planning Calendar** - Plan weekly meals by dragging recipes into a calendar view.

- [ ] **Image Upload** - Let users upload their own photos instead of relying on URLs.

- [ ] **Recipe Yield Tracking** - Remember actual servings vs. planned servings for adjustments next time.

- [ ] **Export/Backup** - Export all recipes as JSON or PDF cookbook.

- [ ] **Recent Recipes** - Quick access to recipes you've made recently.
